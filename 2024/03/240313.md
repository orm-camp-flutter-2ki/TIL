### abstract class


> 추상 클래스는 일부 메서드의 구현을 제공하지 않고, 메서드의 시그니처만을 정의


```dart
abstract class Animal {
  void makeSound();
  void eat();
}
```

</br>
</br>

### interface

> 인터페이스는 클래스가 특정 메서드나 속성을 제공한다는 것을 선언하는데 사용됨 Dart에서는 클래스가 인터페이스를 직접 구현
```dart
abstract interface class Vehicle {
  void start();
  void stop();
}
```
