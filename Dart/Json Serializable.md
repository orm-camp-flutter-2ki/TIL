## 📖 Json Serializable
<br>

- 장점 dto 안쓸 때 데이터를 맵핑할때 씀 (확실할때만)

```dart
 dart pub add json_annotation dev:build_runner dev:json_serializable
 ```
- pub add 해서 다운로드
- dart run build_runner build  (watch)  
- dart run build_runner build --delete-conflicting-outputs (충돌 해결)
