# 📖 연산자
<br>

## 📄 산술 연산자

 - \+ : 더하기
 - \- : 빼기
 - \* : 곱하기
 - \/ : 나누기
 - ~/ : 목
 - % : 나머지
<br>
<br>

## 📄 증감 연산자

- num++ 나중에 계산
- ++num 먼저 계산
<br>
<br>

## 📄 비교 연산자

- == : 같다
- != : 다르다
- \> : 더 크다
- \< : 더 작다
- \>= : 크거나 같다
- <= : 작거나 같다
<br>
<br>

## 📄 논리 연산자

- && : 그리고
- || : 또는
- \== : 같다
- ! : 부정
- != : 다르다
<br>
<br>

## 📄 타입 검사

- is : 같은 타입이면 true
- is! : 다른 타입이면 true
