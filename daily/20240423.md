### Stream
- 실시간으로 생성되고 전송되는 데이터의 흐름

	ex. firebase (remote db)

- 원하는 부분만 UI 갱신이 가능함
- 스트림 빌더 (Future bulder와 사용방법 유사함)

class ImageCard extends StatelessWidget {

      child: GestureDetector(

        child: ClipRRect(

          child: Image.network(photo.url, fit: BoxFit.cover),

  }
}
