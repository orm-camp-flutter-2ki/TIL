Dart 언어
=============
### What is Dart?
[Dart 공식 페이지 - 언어 소개](https://dart-ko.dev/language)   

<br/> 

**다트(Dart)**  
- 객체지향 프로그래밍 언어이다.  
- 타입 추론 기능, 널 세이프티, 비동기 프로그래밍 등.. 강력한 기능 제공한다.  
- 네이티브 크로스 플랫폼 개발에 적합하다.
- 일급객체를 지원하는 언어이다.

<br/>  

**다른 언어와의 중요한 차이**  
- Null Safety  
- Type Safety  
- Hybrid Compile
