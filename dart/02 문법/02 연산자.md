연산자
=============
### 목차  
1. 산술 연산자
2. 증감 연산자
3. 비교 연산자
4. 논리 연산자  

<br/>

## 산술 연산자  
> ```
> + : 더하기   
> - : 빼기  
> * : 곱하기  
> / : 나누기 (double 타입 반환)  
> ~/ : 몫 (int 타입 반환)  
> % : 나머지 (int 타입 반환)  
> ```  

<br/>

## 증감 연산자
> ```
> int num = 0; // 각 코드에 대입 시 계속 0으로 초기화 된다 가정
> print(num++); //  0 출력, 코드 실행 후 증가  
> print(++num); //  1 출력, 증가 후 코드 실행  
> print(num--); //  0 출력, 코드 실행 후 증가  
> print(--num); // -1 출력, 감소 후 코드 실행  
> ```
>✔️ 값은 1씩 증가, 감소한다  
> ```
> num = num + 2;
> num += 2;
> ```

## 비교 연산자  
> ```
> == : 같다  
> != : 다르다  
> >, < : 크거나 작다 (초과, 미만)  
> >=, <= : 크거나 같다 혹은 작거나 같다 (이상, 이하)
> ```

<br/>

## 논리 연산자
> ```
> && : 그리고  
> || : 또는  
> == : 같다  
> ! : 부정  
> != : 다르다  
> ```
