# 상속 (inheritance)
- 이전에 만든 클래스와 닮았지만 일부 다른 클래스를 만들 필요가 있을 경우에 사용한다.
- 기반이 되는 클래스를 `extends`로 ... **기능을 확장하는 것**이다.
- 다중상속은 Dart에서 사용하지 않는다.
