# 제목 없음

> ui = f(stste)
> 
- ui : the layout on the screen
- f : your build methods
- state : 위젯의 상태, 변수 : the application state
    - 변수, state를 별도로 관리하자

```dart
상태 관리란?
상태 = 데이터 = 변수
변수를 수정하면 알아서 UI도 바뀌게 하자
```

## 상태의 종류

### 임시 상태

> 구지 viewmodel로 옮길 필요가 없다
> 
- pageview의 index
- BottomNavigationView의 index
- 애니메이션 상태
