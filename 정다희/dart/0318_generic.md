# 제네릭(Generic)
- 타입을 나중에 원하는 형태로 정의할 수 있음
- 타입 안전 효과
```dart

// 이용 가능한 타입에 제한두기
//-> 이렇게 하는 이유 ? Pocket 클래스 내에서 사용되는 E 타입의 데이터가 Book 클래스의 속성이나 메서드를 가지고있음을 보장하기 위함. 
class Pocket<E extends Book> {
  E? _data;

  void put(E data) {
    _data = data;
  }

  E? get() {
    return _data;
  }
}
```
